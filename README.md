# Декомпозиция интерфейса

Простое React-приложение для демонстрации декомпозиции интерфейса новостной страницы на компоненты.

## Функции

- Отображение верхней панели с новостями и промо
- Поисковая секция с логотипом и ссылками на сервисы
- Блок виджетов (погода, наиболее посещаемые, карта Германии, ТВ-программа, прямой эфир)
- Базовая разметка и стилизация для видимости блоков

## Демо

Посмотреть демо: [GitHub Pages](https://marl3ndev.github.io/decomposition/)

## Начало работы

1. Установите зависимости: `npm install`
2. Запустите приложение: `npm start`
3. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Сборка

Запустите `npm run build` для сборки приложения в продакшн.
